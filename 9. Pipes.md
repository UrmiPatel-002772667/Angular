# Angular Pipes

**Pipes** are a powerful feature in Angular used to **transform and format data directly in templates**.

A pipe **takes input data, processes it, and returns a transformed output** without modifying the original data.

Example:

```html
{{ name | uppercase }}
```

Here the `uppercase` pipe transforms the text to uppercase.

Important points:

* Pipes **only transform data for display**
* They **do not modify the original object**
* They help keep templates **clean and readable**

---

# When to Use Pipes

Pipes are commonly used to:

* Format data for display
* Convert data types
* Perform calculations
* Filter or sort data
* Transform data before displaying it in the UI

---

# Built-in Pipes

Angular provides several **built-in pipes**.

### String Pipes

| Pipe            | Purpose                     |
| --------------- | --------------------------- |
| `UpperCasePipe` | Converts text to uppercase  |
| `LowerCasePipe` | Converts text to lowercase  |
| `TitleCasePipe` | Converts text to title case |

Example:

```html
{{ angularPipes | uppercase }}
{{ angularPipes | lowercase }}
{{ angularPipes | titlecase }}
```

---

### Date Pipe

Formats date values.

```html
{{ currentDate | date:'short' }}
{{ currentDate | date:'mediumDate' }}
{{ currentDate | date:'shortTime' }}
```

Example chaining pipes:

```html
{{ currentDate | date | uppercase }}
```

---

### Number Pipes

| Pipe           | Purpose                       |
| -------------- | ----------------------------- |
| `DecimalPipe`  | Formats numbers               |
| `PercentPipe`  | Converts number to percentage |
| `CurrencyPipe` | Formats numbers as currency   |

Examples:

```html
{{ 3.141592653 | number:'1.3' }}
{{ 0.25 | percent }}
{{ 20.25 | currency }}
{{ 20.25 | currency:'INR' }}
```

---

### JSON Pipe

Converts an object into a JSON string.

```html
{{ person | json }}
```

Useful for **debugging data**.

---

### Slice Pipe

Extracts a portion of an array or string.

Example:

```html
{{ angularPipes | slice:4:8 }}
{{ angularPipes | slice:6:-1 }}
```

---

### Async Pipe

The **AsyncPipe** automatically subscribes to asynchronous data like:

* Observables
* Promises

It also **unsubscribes automatically**, preventing memory leaks.

Example:

```html
<ul>
  @for(i of items | async; track i){
    <li>{{ i }}</li>
  }
</ul>
```

# Custom Pipes

Developers can create **custom pipes** when built-in pipes are not enough.

## Steps to Create a Custom Pipe

1. Create a pipe class
2. Implement `PipeTransform`
3. Write transformation logic
4. Declare the pipe
5. Use it in templates

Example:

```ts
import { Pipe, PipeTransform } from '@angular/core';
import { DatePipe } from '@angular/common';

@Pipe({
  name: 'custom'
})
export class CustomPipe implements PipeTransform {

  private datePipe = new DatePipe('en-US');

  transform(value: any): any {

    if (!value) {
      return 'NA';
    }

    return this.datePipe.transform(value, 'dd-MMM-yyyy');
  }

}
```

Usage:

```html
{{ userDate | custom }}
```

# Types of Pipes

Angular pipes are divided into two types.

---

# 1. Pure Pipes

Pure pipes are executed **only when the input value changes**.

They are **more efficient and optimized for performance**.

Example:

```ts
@Pipe({
  name: 'pure'
})
export class PurePipe implements PipeTransform {

  transform(value: number): number {
    return value * value;
  }

}
```

Usage:

```html
{{ 5 | pure }}
```

Output:

```
25
```

---

# 2. Impure Pipes

Impure pipes are executed **during every change detection cycle**, even if the input value has not changed.

They are **less efficient and should be used carefully**.

Example:

```ts
@Pipe({
  name: 'impure',
  pure: false
})
export class ImpurePipe implements PipeTransform {

  transform(value: any): any {
    return new Date().toLocaleTimeString();
  }

}
```

Usage:

```html
{{ '' | impure }}
```

This will update **every change detection cycle**.
