# TypeScript
- strongly typed programming language
- created and maintained by Microsoft
- superset of javascript
- typescript compiles down to javascript
- supported by all major libraries and framework
- keeps your code free from undefined and null values

## Single Page Application 
- can use Angular, React, or Vue to create SPA
- doesn't make requests to server for every URL request
- Angular has routing functionalities to create SPA. It also Offers Server Side Rendering

## Data Types
- String `let lanem = "Urmi"`
- Number `let age: number`
- Boolean - `let isValid = false`
- Array - `let arr : string[] = ["apple", "banana", "cherry"]` `let numlist : Array<number> `
- Enum - `enum color { Red, Green, Blue }`  `let c: color = color.Green`
- tuple - `let tuple: [number, number] = [40, 30]`
- any - `let department: any`
- void - `function logMessage(message: string): void {}`
- never - ``

## Functions
- filter - filters data 
- find - find a single value from array
- reduce - adds the value till it reaches end of an array
